Detailed Description

Single trace to be plotted.

Application code shouild not attempt to construct a trace Rather call one of plot::AddPointTrace, plot::AddRealTimeTrace or plot::AddStaticTrace which return a reference to the trace which can be configured and be populated with data.

    form fm;
     construct plot to be drawn on form
    plot::plot thePlot( fm );
     construct plot trace
    auto t1 = thePlot.AddStaticTrace();
     provide some data for  trace
    std::vector< double > d1 { 10, 15, 20, 25, 30, 25, 20, 15, 10 };
    t1.set( d1 );
     plot in blue
    t1.color( colors::blue );
     show and run
    fm.show();
    exec();
