\doxysubsection{Detailed Description}
Maintain indices of circular buffer. 

When the buffer is full, new data over-\/writes the oldest ( wraps around ) The data is stored in an external buffer Wrap around is done by manipulating the indices, rather than actually moving the buffer contents
