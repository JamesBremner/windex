\doxysubsection{Detailed Description}
A widget that user can click to toggle a true/false value. 

This draws a custom checkbox that expands with the height of the widget ( set by \mbox{\hyperlink{classwex_1_1gui_a44fb9d06fdb73260e3f155c4cac6886c}{move()}} ) ( The native checkbox is very small and its size cannot be changed )
