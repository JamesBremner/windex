Update ( redraw ) property child widgets. 

Some widgets update nicely when the window containing the property grid resizes Others, most significantly labels, do not. So application code should handle containing window resizes by calling \mbox{\hyperlink{classwex_1_1property_grid_a7982de0d0ce0a9480c603872e11e93f2}{property\+Grid\+::update()}} which will call this method on all properties.
