\doxysubsection{Detailed Description}
A read only table of values. 

The values are displayed in rows of columns. The first column is assumed to be the row index. The row index is returned when the row is clicked.

Usage\+: 
\begin{DoxyPre}
     std::vector<std::vector<std::string>> table1data\{
        \{"{}1"{},"{}a"{}, "{}b"{}, "{}c"{}\},
        \{"{}2"{},"{}x"{}, "{}y"{}, "{}z"{}\}\};
    table \&table1 = maker::make<table>(form);
    table1.move(10, 10, 350, 100);
    table1.set(table1data);
    form.events().asyncReadComplete(
    [\&](int id)
    \{
        std::cout <<"{}row "{} << id << "{}\(\backslash\)n"{};
    \});
\end{DoxyPre}
