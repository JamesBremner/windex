{\footnotesize\ttfamily void wex\+::layout\+::grid (\begin{DoxyParamCaption}\item[{int}]{cols }\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}



Specify number of cols to use for layout. 

The child windows will be laid out in the specified number of columns with the required number of rows to show them all.

The space between columns and rows will be adjusted so that all child windows will be visible.

The size of the child windows will not be altered.
